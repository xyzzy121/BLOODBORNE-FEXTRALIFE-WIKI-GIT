            World Information / Enemies
              +

            Bosses are enemies in Bloodborne who feature a larger health bar and may be defeated to consider a stage "cleared". They will usually drop unique items. Players summoned to help defeat a host's Boss will be returned to their own game world after that Boss has been cleared. Bosses listed below are currently known and more will be added when data becomes available. It isn't 100% clear if the Root Chalice Dungeons will pull Bosses at random from the game or if it has its own set of them.

            Using a series of glitches and exploits, it is possible to beat the game and enter New Game + while only killing Rom and Mergo’s Wet Nurse

            Bosses

             

            Bosses are unique Enemies in Bloodborne. There are 17 normal Bosses, 21 Chalice Dungeon Bosses, and 5 DLC Bosses.

             

            Amygdala
            Location Nightmare Frontier, Defiled Chalice
            Optional Yes
            Drops Ailing Loran Chalice, Great Pthumeru Ihyll Chalice
            Weaknesses Arcane, Fire, Bolt



            Blood Starved Beast
            Location Old Yharnam, Hintertomb Chalice, Ailing Loran Chalice
            Optional Yes
            Drops Pthumeru Chalice, Lower Hintertomb Chalice, Ailing Loran Root Chalice, FRC Loran
            Weaknesses Fire and Serration



            Celestial Emissary
            Location Upper Cathedral Ward
            Optional Yes
            Drops Communion
            Weaknesses Thrust, Bolt and Fire

             

            Cleric Beast
            Location  Central Yharnam
            Optional Yes
            Drops Sword Hunter Badge
            Weaknesses Fire and Serration



            Darkbeast Paarl
            Location Old Yharnam, Lower Loran Chalice
            Optional Yes
            Drops Spark Hunter Badge, Lower Loran Chalice
            Weaknesses Arcane, Fire and Serration



            Ebrietas, Daughter of the Cosmos
            Location Altar of Despair
            Optional Yes
            Drops Great Isz Chalice
            Weaknesses Thrust, Bolt and Fire

             

            Father Gascoigne
            Location Central Yharnam
            Optional No
            Drops Oedon Tomb Key
            Weaknesses Serration



            Gehrman, The First Hunter
            Location Hunter's Dream
            Optional No, except if going for "Yharnam Sunrise" ending
            Drops Old Hunter Badge
            Weaknesses Arcane, Bolt and Fire



            Martyr Logarius
            Location Forsaken Cainhurst Castle
            Optional Yes
            Drops Crown of Illusions
            Weaknesses

            Use The Old Hunter Bone to quicken parry

             

            Mergo's Wet Nurse
            Location Nightmare of Mensis
            Optional No
            Drops Third Umbilical Cord
            Weaknesses Arcane, Fire, Bolt and Rapid Poison



            Micolash, Host of the Nightmare
            Location Mergo's Loft Middle
            Optional No
            Drops Mensis Cage
            Weaknesses None



            Moon Presence
            Location Hunter's Dream
            Optional Yes, except if going for "Childhood's Beginning" ending
            Drops Blood Echoes (230000)
            Weakness Arcane, Fire and Bolt

             

             

            One Reborn
            Location Yahar'gul Chapel
            Optional No
            Drops Yellow backbone x3
            Weaknesses Bolt and Fire



            Rom, the Vacuous Spider
            Location Moonside Lake
            Optional No
            Drops Kin Coldblood(12) x1
            Weaknesses Bolt, Fire, Arcane



            Shadow of Yharnam
            Location Forbidden Woods
            Optional No
            Drops Blood Rapture
            Weaknesses Arcane and Bolt

             

            Vicar Amelia
            Location Cathedral Ward
            Optional No
            Drops Gold Pendant
            Weaknesses Fire and Serration



            Witch of Hemwick
            Location Hemwick Charnel Lane
            Optional Yes
            Drops Bloodshot Eyeball x4
            Weaknesses None
